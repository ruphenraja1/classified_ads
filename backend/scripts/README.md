# Backend Scripts

This directory contains utility scripts and custom functionalities for the Django backend.

## Files

- `advanced_verify.py` - Advanced verification logic
- `check_lov_count.py` - Script to check LOV (List of Values) counts
- `create_home_ui_lov.py` - Creates LOV data for home UI
- `create_test_data.py` - Generates test data
- `create_user_message_lov.py` - Creates LOV data for user messages
- `import_all_cities.py` - Imports city data
- `insert_cities.py` - Inserts city records into database
- `verify_import_data.py` - Verifies imported data integrity

## Usage

These scripts are typically run manually or as part of data migration/setup processes. Some may be called by Django management commands.
